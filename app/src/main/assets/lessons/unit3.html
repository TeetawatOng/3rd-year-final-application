ตัวดำเนินการ (Operators) ใน JavaScript เป็นสัญลักษณ์ที่ใช้ในการดำเนินการกับตัวแปรและค่าต่างๆ โดยสามารถแบ่งออกเป็นประเภทหลักๆ ได้ดังนี้:

1. ตัวดำเนินการทางคณิตศาสตร์ (Arithmetic Operators)

+ (บวก) เช่น 5 + 3 = 8
- (ลบ) เช่น 5 - 3 =2
* (คูณ) เช่น 5 * 3 = 15
/ (หาร) เช่น 5 / 2 = 2.5
% (หารเอาเศษ) เช่น 5 % 2 = 1
** (ยกกำลัง) เช่น 5 ** 2 = 25>

2. ตัวดำเนินการทางตรรกะ (Logical Operators)

&& (AND (จริงเมื่อทั้งสองเป็น true) เช่น true && false //false
` เช่น ` //OR (จริงเมื่ออย่างน้อยหนึ่งเป็น true)
! (NOT (กลับค่าความจริง)) เช่น !true //false

3. ตัวดำเนินการเปรียบเทียบ (Comparison Operators)

== (เท่ากัน (ไม่สนใจชนิดข้อมูล)) เช่น 5 == "5" //true
=== (เท่ากันแบบเคร่งครัด (รวมชนิดข้อมูล)) เช่น 5 === "5" //false
!= (ไม่เท่ากัน) เช่น 5 != 3 //true
!== (ไม่เท่ากันแบบเคร่งครัด) เช่น 5 !== "5" //true
> (มากกว่า) เช่น 5 > 3 //true
< (น้อยกว่า) เช่น 5 < 3 //false
>= (มากกว่าหรือเท่ากับ) เช่น 5 >= 5 //true
<= (น้อยกว่าหรือเท่ากับ) เช่น 5 <= 2 //false

4. ตัวดำเนินการกำหนดค่า (Assignment Operators)

= (กำหนดค่า) เช่น x = 5 //x = 5
+= (เพิ่มค่าและกำหนด) เช่น x += 3 //x = x + 3
-= (ลดค่าและกำหนด) เช่น x -= 2 //x = x - 2
*= (คูณและกำหนด) เช่น x *= 2 //x = x * 2
/= (หารและกำหนด) เช่น x /= 2 //x = x / 2
%= (หารเอาเศษและกำหนด) เช่น x %= 2 //x = x % 2

5. ตัวดำเนินการเพิ่มค่าและลดค่า (Increment & Decrement Operators)

++ (เพิ่มค่า 1) เช่น x = 5; x++ //x = 6
-- (ลดค่า 1) เช่น x = 5; x-- //x = 4
