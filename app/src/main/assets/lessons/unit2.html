ตัวแปร (Variable) ใน JavaScript คือ พื้นที่เก็บข้อมูล ที่ใช้เก็บค่าต่าง ๆ เช่น ตัวเลข (Number), ข้อความ (String), ค่า Boolean (true/false), อาร์เรย์ (Array) หรือออบเจ็กต์ (Object) เพื่อใช้ในการประมวลผลของโปรแกรม เมื่อเราประกาศตัวแปร เราสามารถกำหนดค่าให้มัน และสามารถนำค่าที่เก็บไว้นั้นไปใช้ในโค้ดได้

1. var

คุณสมบัติของ let
1. มี Function Scope (ใช้ได้ทั้งฟังก์ชันที่มันอยู่ แต่ไม่จำกัดอยู่แค่บล็อก {})
2. สามารถประกาศซ้ำ (re-declare) ได้
3. มี Hoisting (ตัวแปรจะถูกยกขึ้นไปก่อนรันโค้ด แต่ค่าเริ่มต้นเป็น undefined)

var x = 10;
console.log(x); // 10

2. let

คุณสมบัติของ let
1. มีBlock Scope (ใช้ได้แค่ใน {} ที่ประกาศไว้)
2. ไม่สามารถประกาศซ้ำ (re-declare) ได้ใน block เดียวกัน
3. มี Hoisting แต่ไม่สามารถใช้ตัวแปรก่อนประกาศได้

let y = 20;
console.log(y); // 20

3. const

คุณสมบัติของ const
1. มี Block Scope เหมือน let
2. ต้องกำหนดค่าตั้งแต่เริ่มต้น และค่าห้ามเปลี่ยน (immutable)
3. ใช้กับค่าคงที่ เช่น ค่าทางคณิตศาสตร์ (const PI = 3.1416;)

const z = 50;
console.log(z); // 50

4. array

คุณสมบัติของ const
1. อาร์เรย์สามารถเก็บค่าหลายค่าในตัวแปรเดียว
2. อาร์เรย์ใช้ดัชนี (Index) ในการเข้าถึงค่า
3. อาร์เรย์สามารถมีข้อมูลประเภทต่างกันได้

let numbers = [10, 20, 30, 40, 50];
console.log(numbers); // Output: [10, 20, 30, 40, 50]
----------------------------
let colors = ["Red", "Green", "Blue"];
console.log(colors[0]); // Output: "Red"
